POST /proofreq


Relevant Schemas

ProofReq is the protobuff in the request body and follows:
message ProofReq {
  required bytes uid = 1;
  required bytes unonce = 2;
  required int64 seqid = 3;
  required bytes vid = 4;
  required bytes sig = 5;
}


ProofResp is the protobuff in the response and follows:
message ProofResp {
  required bytes uid = 1;
  required bytes unonce = 2;
  required bytes sig = 3;
}

Request:
* Supported content types are:
o application/x-protbuf
